When you set up an alert for your site, you will be notified when there is an issue with one or more on-ramps. These alerts are sent when the percentage of successful health checks for a {props.productName} on-ramp drops below the selected service-level objective (SLO). Setting health alerts will also display unhealthy tunnels in the Network map and in the Traffic overview sections.

To set up health alerts:

1. Configure <a href={props.magicTunnelHcsURL}>Tunnel health alerts</a> across all of the tunnels associated with each {props.productName} site.
2. After configuring Tunnel health alerts, any {props.productName} site with a tunnel (on-ramp) that is outside of its SLO threshold will be labeled unhealthy in Network map and Traffic overview.
