Cloudflare WAN (formerly Magic WAN) provides site-to-site connectivity and packet-level security through [Cloudflare Network Firewall](/cloudflare-network-firewall/). The Cloudflare One integrations below extend that foundation with identity-aware security — user-level access policies through [Cloudflare Access](/cloudflare-one/access-controls/), secure Internet egress through [Cloudflare Gateway](/cloudflare-one/traffic-policies/), and device-level routing through the [Cloudflare One Client (formerly WARP)](/cloudflare-one/team-and-resources/devices/cloudflare-one-client/).

Review the tutorials to learn how to use Cloudflare WAN with these Cloudflare One products.
