---
title: New optimization features in Images
description: >
  New composite modes, percentage-based overlay dimensions, fit modes, and AI upscaling.
date: 2026-06-16
---

These updates introduce new features for optimizing and manipulating with Images:

- **New `composite` option:** Control how [overlays are blended](/images/optimization/draw-overlays/#composite) with the base image.
- **Percentage widths:** Set the dimensions of an overlay as [a fraction of the dimensions](/images/optimization/draw-overlays/#width-and-height) of the base image.
- **New `fit` modes:** Use [`aspect-crop`](/images/optimization/features/#aspect-crop) to always preserve the target aspect ratio or [`scale-up`](/images/optimization/features/#scale-up) to always enlarge images.
- **New `upscale` parameter:** Apply [AI upscaling](/images/optimization/features/#upscale) to produce sharper, more detailed results when enlarging images.
