[**Radar**](/radar/) now includes a [Network Quality Test](https://radar.cloudflare.com/speedtest) page. The tool measures Internet connection quality and performance, showing connection details such as IP address, server location, network (ASN), and IP version. For more detailed speed test results, the page links to [speed.cloudflare.com](https://speed.cloudflare.com/).
