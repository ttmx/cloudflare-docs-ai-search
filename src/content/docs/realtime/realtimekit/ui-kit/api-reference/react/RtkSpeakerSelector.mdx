A component which lets to manage your audio devices and audio preferences.
 Emits `rtkStateUpdate` event with data for muting notification sounds:
 ```ts
 {
  prefs: {
    muteNotificationSounds: boolean
  }
 }
 ```

## Properties

| Property | Type | Required | Default | Description |
|----------|------|----------|---------|-------------|
| `iconPack` | `IconPack` | ÔŁî | `defaultIconPack` | Icon pack |
| `meeting` | `Meeting` | Ôťů | - | Meeting object |
| `size` | `Size` | Ôťů | - | Size |
| `states` | `States` | Ôťů | - | States object |
| `t` | `RtkI18n` | ÔŁî | `useLanguage()` | Language |
| `variant` | `'full' \| 'inline'` | Ôťů | - | variant |

## Usage Examples

### Basic Usage

```tsx
import { RtkSpeakerSelector } from '@cloudflare/realtimekit-react-ui';

function MyComponent() {
  return <RtkSpeakerSelector />;
}
```

### With Properties

```tsx
import { RtkSpeakerSelector } from '@cloudflare/realtimekit-react-ui';

function MyComponent() {
  return (
    <RtkSpeakerSelector
      meeting={meeting}
      size="md"
      variant={'full' | 'inline'}
    />
  );
}
```
