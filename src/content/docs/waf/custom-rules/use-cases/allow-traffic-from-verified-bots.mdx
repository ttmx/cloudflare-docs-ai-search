This example [custom rule](/waf/custom-rules/create-dashboard/) challenges requests from a list of countries, but allows traffic from search engine bots — such as Googlebot and Bingbot — and from other [verified bots](/bots/concepts/bot/verified-bots/).

The rule expression uses the [`cf.client.bot`](/ruleset-engine/rules-language/fields/reference/cf.client.bot/) field to determine if the request originated from a known good bot or crawler.

- **When incoming requests match**:

  | Field      | Operator | Value                     | Logic |
  | ---------- | -------- | ------------------------- | ----- |
  | Country    | is in    | `Mexico`, `United States` | And   |
  | Known Bots | equals   | `false`                   |       |

  If you are using the expression editor:<br/>
  `(ip.src.country in {"US" "MX"} and not cf.client.bot)`

- **Then take action**: _Managed Challenge_
