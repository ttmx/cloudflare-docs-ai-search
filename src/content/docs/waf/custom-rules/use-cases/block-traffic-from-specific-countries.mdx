This example [custom rule](/waf/custom-rules/create-dashboard/) blocks requests based on country code using the [`ip.src.country`](/ruleset-engine/rules-language/fields/reference/ip.src.country/) field.

- **When incoming requests match**:

  | Field   | Operator | Value                   |
  | ------- | -------- | ----------------------- |
  | Country | is in    | `Korea, North`, `Syria` |

  If you are using the expression editor:<br/>
  `(ip.src.country in {"KP" "SY"})`

- **Then take action**: _Block_
