This example [custom rule](/waf/custom-rules/create-dashboard/) blocks requests based on country code using the [`ip.src.country`](/ruleset-engine/rules-language/fields/reference/ip.src.country/) field, only allowing requests from two countries: United States and Mexico.

- **When incoming requests match**:

  | Field   | Operator  | Value                     |
  | ------- | --------- | ------------------------- |
  | Country | is not in | `Mexico`, `United States` |

  If you are using the expression editor:<br/>
  `(not ip.src.country in {"US" "MX"})`

- **Then take action**: _Block_
