You can configure alerts for resources detected in your domain. Refer to [Alerts](/client-side-security/alerts/) for more information.

## New resource alerts

:::note
Requires a Business plan or higher.
:::

New resource alerts notify you about new resources detected on your domain, resources detected from new host domains, or issues with the URL length of newly detected resources.

<AvailableNotifications
	product="Client-side security"
	notificationFilter="Client-side security New Resources Alert"
/>
<AvailableNotifications
	product="Client-side security"
	notificationFilter="Client-side security New Domain Alert"
/>
<AvailableNotifications
	product="Client-side security"
	notificationFilter="Client-side security New Resource Exceeds Max URL Length Alert"
/>

## Code change alert

:::note
Only available to customers with Client-Side Security Advanced.
:::

This alert notifies you about [code changes](/client-side-security/detection/review-changed-scripts/) in previously detected scripts.

<AvailableNotifications
	product="Client-side security"
	notificationFilter="Client-side security New Code Change Detection Alert"
/>

## Malicious resource alerts

:::note
Only available to customers with Client-Side Security Advanced.
:::

Malicious resource alerts notify you about [resources considered malicious](/client-side-security/how-it-works/malicious-script-detection/), based on their [domain](/client-side-security/how-it-works/malicious-script-detection/#malicious-domain-checks), [URL](/client-side-security/how-it-works/malicious-script-detection/#malicious-url-checks), or [script content](/client-side-security/how-it-works/malicious-script-detection/#malicious-script-detection).

<AvailableNotifications
	product="Client-side security"
	notificationFilter="Client-side security New Malicious Domain Alert"
/>
<AvailableNotifications
	product="Client-side security"
	notificationFilter="Client-side security New Malicious URL Alert"
/>
<AvailableNotifications
	product="Client-side security"
	notificationFilter="Client-side security New Malicious Script Alert"
/>

Malicious resource alerts will only include resources with an _Active_ status. Refer to [Script and connection statuses](/client-side-security/reference/script-statuses/) for more information.
