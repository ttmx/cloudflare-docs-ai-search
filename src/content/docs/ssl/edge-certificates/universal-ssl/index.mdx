<GlossaryDefinition term="Universal SSL certificate" />

On a [full setup](/dns/zone-setups/full-setup/), Universal SSL certificates cover your root domain (for example, `example.com`) and first-level subdomains (for example, `www.example.com`). On a [partial (CNAME) setup](/dns/zone-setups/partial-setup/), each proxied subdomain receives its own certificate regardless of depth. Cloudflare handles issuance, renewal, and deployment automatically.

For full setup zones that need coverage beyond first-level subdomains, use [Total TLS](/ssl/edge-certificates/additional-options/total-tls/) or [advanced certificates](/ssl/edge-certificates/advanced-certificate-manager/).

Universal certificates are [Domain Validated (DV)](/ssl/concepts/#validation-level), which means the certificate authority verifies domain ownership but does not validate organization identity. For setup details, refer to [Enable Universal SSL](/ssl/edge-certificates/universal-ssl/enable-universal-ssl/).

:::note
Universal SSL certificates are issued after your domain is active on Cloudflare. If you need an SSL certificate before migrating traffic, or if you need to [customize cipher suites](/ssl/edge-certificates/additional-options/cipher-suites/customize-cipher-suites/), use [Advanced](/ssl/edge-certificates/advanced-certificate-manager/) or [Custom](/ssl/edge-certificates/custom-certificates/) certificates.
:::
