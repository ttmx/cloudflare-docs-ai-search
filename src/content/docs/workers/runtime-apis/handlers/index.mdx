Handlers are methods on Workers that can receive and process external inputs, and can be invoked from outside your Worker. For example, the `fetch()` handler receives an HTTP request, and can return a response:

```js
export default {
	async fetch(request, env, ctx) {
		return new Response('Hello World!');
	},
};
```

## Handlers in Python Workers

When you [write Workers in Python](/workers/languages/python/), handlers are placed in a class named `Default` that extends the [`WorkerEntrypoint` class](/workers/runtime-apis/bindings/service-bindings/rpc/) (which you can import from the `workers` SDK module).
