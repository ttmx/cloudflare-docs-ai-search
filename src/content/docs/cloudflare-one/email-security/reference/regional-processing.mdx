Email security uses Cloudflare's [Data Localization Suite (DLS)](/data-localization/) to allow you to control where your emails are processed. You do not need Data Localization Suite with Email security to choose the different locations.

:::note
Regional processing is only available for customers who deploy Email security via MX/Inline or BCC/Journaling.
:::

Currently, you can process emails in the following regions:

- US
- Germany
- India
- Australia
