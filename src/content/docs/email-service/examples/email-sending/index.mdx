Advanced patterns and examples for sending emails with Cloudflare Email Service. Most examples use the [Workers binding](/email-service/api/send-emails/workers-api/); the same [`EmailMessageBuilder`](/email-service/api/send-emails/workers-api/#send-method) fields (`to`, `from`, `subject`, `html`, `text`, `cc`, `bcc`, `replyTo`, `attachments`, `headers`) apply to the [REST API](/email-service/api/send-emails/rest-api/) as JSON in your HTTP request body. The [Send email over SMTP](/email-service/examples/email-sending/smtp/) example covers sending over SMTP from several languages and clients.
